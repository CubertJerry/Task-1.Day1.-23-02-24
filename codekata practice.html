codekata practice Code kata practice is an excellent way to sharpen your coding skills and improve problem-solving abilities. Here's a simple problem for you to tackle: Problem: FizzBuzz Write a function that takes a number as input and returns an array of strings. For each number from 1 to the input number, the function should append one of the following strings to the array: "Fizz" if the number is divisible by 3. "Buzz" if the number is divisible by 5. "FizzBuzz" if the number is divisible by both 3 and 5. Otherwise, the number itself as a string. Example: Input: 5 Output: ["1", "2", "Fizz", "4", "Buzz"] function fizzBuzz(n) { const result = []; for (let i = 1; i <= n; i++) { if (i % 3 === 0 && i % 5 === 0) { result.push("FizzBuzz"); } else if (i % 3 === 0) { result.push("Fizz"); } else if (i % 5 === 0) { result.push("Buzz"); } else { result.push(i.toString()); } } return result; } // Test the function const input = 5; const output = fizzBuzz(input); console.log(output); // Output: ["1", "2", "Fizz", "4", "Buzz"]
